How to work in this program

Icon Description
Save current state of canvas into PNG file.
Clear the canvas.
Open settings window for chosen tool.
Straight line tool.
Curve line tool.
Fill area with a color.
Eraser tool.
Drawing triangles with chosen color and line thickness.
Drawing squares with chosen color and line thickness.
Drawing pentagons with chosen color and line thickness.
Drawing a custom polygon with chosen color, number of angles, rotation angle and line thickness.
Drawing a custom star with chosen color, number of angles, rotation angle and line thickness.
Five default colors for drawing: red, yellow, green, blue and black, last button with name "..." opens the window with palette.
Undo last action. The size of history of changes is 15 actions.
