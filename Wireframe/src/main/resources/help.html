Help

1st window: spline editor

Parameter Description
Segments count (N) The number of lines between the spline control points.
Generatrix count (M) The number of repetitions of the spline to form a shape.
Lines between generatrix lines (M1) The number of lines between generatrix lines (accuracy of circles).
Fields 'x' & 'y' Add control point manually by its coordinates. Type coordinates and press 'Add'.
Apply Apply settings without closing the editor.
OK Apply settings with closing the editor.

2nd window: Wireframe viewer.

Button Description
Edit Show/Hide editor window.
Save Save the spline to a file.
Open Load spline from a file.
Reset Reset zoom and angles of rotation.
